# Cap'n Proto Schema Package

This directory contains the public RPC contracts for Clandes.

Files:

- `clandes.capnp`
- `account.capnp`
- `policy.capnp`
- `proxy.capnp`
- `claude_auth.capnp`
- `claude_query.capnp`
- `codex_auth.capnp`
- `codex_query.capnp`
- `common.capnp`

Releases also include:

```text
clandes-capnp-schemas-<version>.tar.gz
clandes-capnp-schemas-<version>.tar.gz.sha256
```

The archive contains the same `*.capnp` files from this directory.
